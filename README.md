# AAVAIL Revenue Forecasting using Machine Learning

This project forecasts revenue using a simple Linear Regression model served through a Flask API.

## Files Included
- app.py – Flask API for predictions
- model.py – ML model training script
- test_app.py – API unit tests
- test_model.py – Model unit tests
- Dockerfile – Containerization setup
- requirements.txt – Required libraries

## How to Run
1. Clone the repo.
2. Run `python model.py` to train and save the model.
3. Run `python app.py` to start the Flask API.
4. Use `python -m unittest` to run tests.
